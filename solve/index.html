How to Construct the Data File


Download the excel file below. Once there, follow the directions to recreate your puzzle in the excel sheet. Everytime you want to start a new puzzle, this will be the step you start at. Once you've finished, a program will run and a seperate sheet will create the data you need to finish the solver.

⤓ Download Excel File

Important: The AMPL_File_Generator.xlsm was coded to export data to windows systems only. Although, it will collect the data and simulate the light path for Mac users, but it won’t export the data automatically.

Overview of the Data File:

Main areas of the file:
Orange: Area to enter the puzzle data.
Yellow: Area to enter the size of the puzzle.
Blue: IndLit data.

Buttons of the file:
“Run!”: This button starts the light path simulation and collection of data.
“Clear output”: This button clears all the data collected and displayed. This button is there for user’s convenience. Every time “Run” is pressed the previously recorded data is automatically cleared out the memory before the code starts collecting the new data set.
“Export Output”: This button compiles the data collected into a ready to use data file on AMPL.
“Clear Puzzle”: This button erases the information written into the puzzle area.
“Retrieve default puzzle (20x20)”: This button retrieves the default 20x20 puzzle proposed in this project.

The data on the right of the highlighted Blue Area is for user’s reference only. The displayed data makes it easier for the user to double check the white and black boxes’ location on the puzzle area. Any necessary alteration on a black box’s location has to be done in the puzzle area.

1) First, find the puzzle you want to solve on http://brainbashers.com/lightup.asp.
This is the one we choose:

http://brainbashers.com/lightup.asp
Easy 5x5 April 25, 2020

2) Then download and open the excel file labeled “AMPL_data_File_Generator.xlsm”
As soon you will the yellow bar on top of the file asking to Enable Content:

(Windows User)

IMPORTANT: Before starting using the file make sure to Enable Macros.

3) After opening the file, you will see the default puzzle, 20x20, displayed on the puzzle area. To create the data file for that puzzle you just need to click on the button “Run!” and watch the simulation of the light path while the data file is being created for you. Otherwise jump to step 6 to enter your own puzzle.

4) Once the message box shows, the data collection is done, Press “Ok”.

5) Now you can export the data file straight to your computer. Click on the button “Export Output”. Then a Save as window will pop-up. Name the file 'lightupdat.txt' and save it in the same folder as your AMPL files. The extension of the file is fixed as “.txt” by default, that is the file you will need to load into AMPL to get the solution for your puzzle

6) To enter your own puzzle, first you need to click the “Clean Puzzle” button to erase all the information in the puzzle area. The message box bellow will appear, Press “Ok” to continue.

7) After clearing the puzzle area, erase the values of Rows and Columns from previous game, and enter the right size of the puzzle you have chosen In our case, our puzzle is a 5x5, Rows 5, Columns 5:

8) Fill up the empty puzzle area with the data from the puzzle you have chosen. You will have to enter the value of the black boxes into the right row and column. For the black boxes without a number on it, enter “x” to mark the placement of the empty black box.
Note that the rows are indicated by the number on the green cells and the columns by the numbers on the pink cells.
Here is how our chosen puzzle looks in the excel file:

http://brainbashers.com/lightup.asp
Easy 5x5 April 25, 2020

IMPORTANT:Take a time to double check the right entry off all black boxes, one wrong placement of a box will generate the wrong file.

9) After entering your data, press the “Run!” button. Watch the light path simulation while the file compiles all the data for you. Once the message box shows the data collection is done, Press “Ok”.

10) Now you can export the data file straight to your computer. Click on the button “Export Output”. Then a Save as window will pop-up. Name the file 'lightupdat.txt' and save it in the same folder as your AMPL files. The extension of the file is fixed as “.txt” by default, that is the file you will need to load into AMPL to get the solution for your puzzle




How to Run AMPL


Double-click "sw.exe." You will be prompted to enter commands. ctrl + c and ctrl + v the commands below.
Make sure to press Enter after each line.

sw: ampl
apml: option solver cplex;
ampl: option cplex_options 'sensitivity';
ampl: model lightupmod.txt;
ampl: data lightupdat.txt;
ampl: solve;
ampl: display x;
