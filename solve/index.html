How to Construct the Data File


Download the excel file below. Once there, follow the directions to recreate your puzzle in the excel sheet. Everytime you want to start a new puzzle, this will be the step you start at. Once you've finished, a program will run and a seperate sheet will create the data you need to finish the solver. Save the sheet as "lightupdat.txt" and move on to the next step.

⤓ | Download Excel File




How to Run AMPL


Double-click "sw.exe." You will be prompted to enter commands. ctrl + c and ctrl + v the commands below.
Make sure to press Enter after each line.

sw:ampl
apml:option solver cplex;
ampl:option cplex_options 'sensitivity';
ampl:model lightupmod.txt;
ampl:data lightupdat.txt;
ampl:solve;
ampl:display x;
